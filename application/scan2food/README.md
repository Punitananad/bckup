# Scan2food
an application which help the theatre's to serve the food to there costumers onto there seat by just a simple scan.



![Logo](https://scan2food.com/static/assets/images/brand/Scan2Food-whiteBG.png)

## Goal of this project

The goal of this project is very simple which are:

- To create an application where any body can order there food by just sitting only on its seat

- it helps the theatre to manage there staff and the food delivery onto there seats

- it extends the theatre seeling time from 15 min to complete movie time aprox 2-3 hours.

- Theatre didn't have to pitch the customers for the food by going onto there seats.
